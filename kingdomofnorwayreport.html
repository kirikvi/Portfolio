The Semester Project Exam

This is my latest project, and I am happy about the design and about the coding. I prioritized to update the other two projects, and therefore ended up with little time left to update this project and unfortunatly did not have time to ask for help with the carousel.

I made the following updates:

Making a clone

I was terrified to make changes to the actual project exam, so I copied the files manually and created a new folder and made a new repository from scratch to make sure I did not do anything wrong by just cloning the repository with GitHub.

Carousel

I don't know how to fix the "zoom" problem when clicking the next/previous button.

This annoys me, because other than that I am happy about the design. If I could have gone back in time, I would have asked for help on the first day, so I could have been 100% pleased with this project. I would also have asked how to make the carousel go in a loop.

CSS updates

I came across a minor tweak with the archive item images on smaller screens. I had styled the image in the media query with max-width instead of width, which made the image narrower than what I wanted it to be.
