My first ever page, which made me realize I made the right decision starting this programme. I made the following updates:

Adjustments after feedback from tutors

"It's a relatively small thing, but I'd recommend keeping folder names lowercase. So instead of 'Images', just 'images'." - MJ

So i changed this, and therefore had to find every image href to correct the location name.

"The only UI change I would recommend is to change up one of the decorative heading fonts for a simpler one that is easy to read; as they both are currently quite busy. Maybe swopping out the serif type for a cleaner Sans serif (a bold weight of the body typography would work!)" - Suzani Reitz

I changed the serif font to the body font as suggested, and here is the before and after. I think I prefer the serif font, but i would like to keep the handwritten style on the next line, so I kept the change to not have two "busy" fonts next to eachother.

DRY principle

While changing the font, I could not believe how messy my code are. I have not been following the DRY principle, so I did a "clean up" to make the code more well written and made sure I did not repeat myself.

After the changes, the elements having the candara font is only written once, instead of several times. The same goes for other styling, such as text-transform, font-weight and colour. The CSS file went from over 1700 lines to about 1650 lines after the first clean up.

Jeg husker jeg syntes jeg gjorde det helt ok DRY-messig, men når jeg tenker tilbake forstår jeg hvordan jeg endte opp med en så rotere CSS-fil. Underveis mens jeg lagde siden, la jeg til flere og flere nye elementer, og stylet de en og en etterhvert som jeg la de til. Derfor endte jeg dessverre opp med mange repeterte koder, og derav et nett med rot. Dersom jeg skulle laget denne siden på nytt, ville jeg ha lagt til absolutt all htmlen først, for så å style den som en helhet, istedenfor mange individuelle komponenter i en stor masse.

Removed console.log from script files

It is no use for the console.logs now that the site is finished, so I removed them to make it easier to read the files.

The Design

I did not change anything about the design, other than the font I got feedback about. I am happy with the design of the site.
